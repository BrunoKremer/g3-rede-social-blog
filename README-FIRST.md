# Titulo: Reclame Aqui Municipal 
# Status: Meu projeto está em desenvolvimento
# Start Date: 20 de Agosto de 2021
# Descrição: 
# 1.Este projeto permitira que pessoas em minha cidade possam se cadastrar e reclamar de produtos ou serviços tomados localmente e nós utilizaremos nosso suporte juridico para buscar solução junto das empresas reclamadas. 
# 2.Vamos monetizar nossa aplicação oferecendo ranking das empresas reclamadas, produtos reclamados e oferecer suporte juridico para cobranças extra-judiciais ou acordos comerciais.
